Unofficial Connector

Your Garmin Data. Claude Ready.

GarMCP is an unofficial Garmin Connect MCP server for Claude and co. Deploy in minutes on Vercel, connect Garmin, and query your training and health data with AI.

Why GarMCP?

Claude-First Remote MCP

Built for Streamable HTTP and Claude custom connectors, including mobile usage.

Read-Heavy Tooling

97 Garmin tools for activity, sleep, readiness, profile, and long-term trends.

Simple Cloud Stack

Vercel + Upstash Redis REST token storage for lightweight single-user deployments.

How It Works

The complete setup and connector flow are documented on GitHub, including the one-click deploy path, environment variables, and Claude connector configuration.
